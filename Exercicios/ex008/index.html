Principais formatações.

Negrito/ Destaque

Nesta frase temos um termo em negrito usando a tag b (não semantica ou seja sem sentido apenas forma.)

Nesta frase tenho um termo em destaque usando a tag STRONG ( semantica com sentido)

Italico/ Enfâse

Nesta frase temos um termo em italico usando a tag i

Nesta frase temos um termo em enfâse usando a tag EM ( semantica)

Texto marcado.

Podemos criar também um texto marcado usando a tag mark.

Podemos alterar a cor do marca texto a partir so css no caso colocamos a tag style-background-color dentro de mark ai podemos escolher a cor que desejarmos.

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno neste paragrafo.

Texto deletado.

Podemos marcar um texto como excluido para que ele seja lido porém não seja considerado.

Texto inserido.

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois.

Texto sobrescrito

Para inserir coisa do tipo X20+3

Texto subescrito

Para inserir coisas do tipo h2O
